    Getting Started Guides #

    Short guides for BC CIS Students

    What this site is about

    This repository contains notes on getting started guides to various software development tools and concepts for the Brooklyn College CIS undergraduate student. It was started by Professor Chuang as a side project to amplify course notes; it is not an official repository representing the department at the time of this writing.

    Contents of this site are continuously under refinement.
